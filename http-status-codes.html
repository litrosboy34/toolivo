What are HTTP status codes?

HTTP status codes are server responses that explain whether a request succeeded, redirected, failed because of the client or failed because of the server.

What does 404 mean?

404 Not Found means the requested page or resource could not be found on the server.

What does 500 mean?

500 Internal Server Error means the server encountered an unexpected problem while processing the request.
