What is a MIME type?

A MIME type tells browsers, servers and apps what kind of file is being handled, such as image/png, application/json or text/css.

How to use this tool

  • Type a file extension like png, pdf or mp4.
  • Search by MIME type like application/json.
  • Use the result in headers, uploads or web projects.

FAQ

Why are MIME types important?

MIME types help browsers and servers understand how to process, display or download a file correctly.

Can I search by extension and MIME type?

Yes. You can search using extensions like png or pdf, or MIME values like image/png and application/pdf.
