What is a MIME type?

A MIME type tells browsers, servers and apps what kind of file is being handled, such as image/png, application/json or text/css.

How it works

  • Type a file extension like png, pdf or mp4.
  • View the matching MIME type instantly.
  • Use the result in headers, uploads or web projects.
