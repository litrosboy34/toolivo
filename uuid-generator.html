What is a UUID?

UUID stands for Universally Unique Identifier. It is a 128-bit value commonly used to identify records, users, files, requests and resources in software systems.

What is UUID v4?

UUID v4 is a randomly generated UUID version. It is widely used because it does not require a central database or coordination between systems.
