Encode special characters into HTML entities or decode HTML entities back into readable text instantly.

HTML entities are used to safely display reserved characters like <, >, & and quotation marks inside HTML pages.

Common HTML Entities

< becomes &lt;, > becomes &gt;, & becomes &amp;, " becomes &quot; and ' becomes &#39;.

What is HTML entity encoding?

HTML entity encoding converts reserved characters into safe text representations so browsers display them instead of treating them as HTML code.

How to use this tool

  • Paste text or HTML code into the input box.
  • Click Encode to convert characters into entities.
  • Click Decode to convert entities back into readable text.
